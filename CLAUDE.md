## Hard rules

1. **No AI attribution anywhere.** No `Co-Authored-By: Claude`, no `đź¤– Generated with`, no "Generated by", no "AI-assisted" in commits, code, comments, frontmatter, docs, or PR bodies. Strip from any default template that tries to inject it.

They live only in `defense-beacon/reference/out/` which is gitignored. Test fixtures must use obviously fake placeholders such as `AKIA_BEACON_TESTFIXTURE000000000000`.
3. **All commits must be signed AND DCO-signed-off.** Branch protection on `main` enforces both. Use `git commit -S --signoff`. Never `--no-verify`.
4. **PRs only â€” no direct push to `main`.** The repo is public; operator reviews and merges.

## Project-scoped automations

### Hooks (`.claude/settings.json` -> `.claude/hooks/*.sh`)

| Hook | Event | Matcher | What it does |
|---|---|---|---|
| `canary-literal-block.sh` | PreToolUse | `Edit\|Write\|MultiEdit` | Refuses any edit that would commit a real-shaped canary literal anywhere outside `defense-beacon/reference/out/`. Exit 2 with a redacted message naming the canary kind. |
| `run-package-tests.sh` | PostToolUse | `Edit\|Write\|MultiEdit` | When an edit lands inside `defense-beacon/reference/` or `trust-zones/reference/`, runs that package's tests and surfaces failures via exit 2. Feedback only â€” does not block the write itself. |

Hooks resolve the repo root via `${CLAUDE_PROJECT_DIR}`, which Claude Code sets to the project root, so they work for any contributor who clones the repo.

### Subagents (`.claude/agents/*.md`)

| Subagent | When to invoke | Tools |
|---|---|---|
| `swanlake-pre-publish` | Before any release, tag, or push the operator intends to publish. Three parallel scans (attribution drift, real-canary literals, host/path leaks). Returns `publication-clean` or a punch-list. | `Bash, Read, Grep, Glob` (read-only) |
| `threat-model-honesty-reviewer` | Whenever a doc-touching PR is opened or before a release. Lints THREAT-MODEL / NON-GOALS / README / docs for overclaim drift; emits a table. | `Read, Grep, Glob` (read-only) |
